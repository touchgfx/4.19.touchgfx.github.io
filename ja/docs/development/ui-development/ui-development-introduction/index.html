UI開発の概要

機能的なUIの開発は、グラフィックが最適な状態で組み込まれた製品を生むために不可欠な部分であるため、TouchGFXは、高速パフォーマンスだけでなく、円滑な開発エクスペリエンスを提供することを目指しています。

UI開発の章では、TouchGFXアプリケーションの開発方法の詳細に重点を置いています。

  • ソフトウェア･アーキテクチャ - TouchGFXアプリケーション全体のアーキテクチャと設計、およびTouchGFX Designerにより生成されたコードとユーザ･コードの関係について説明します。
  • TouchGFXの操作 - TouchGFXアプリケーションの開発ワークフローと、そのプロセスで使用されるさまざまなツール（PCシミュレータから多数のサポートされたIDEまで）に関する情報が含まれています。
  • Designerのユーザ･ガイド - TouchGFX Designerのさまざまなコンポーネントの使用方法に関する広範なガイドとヒントやコツが含まれています。
  • TouchGFXエンジンの機能 - ビットマップ･キャッシュ、パーシャル･フレームバッファ、多言語サポートなど、TouchGFXエンジンのさまざまな機能に関する情報が含まれています。
  • UIコンポーネント - ウィジェットからコンテナまで、TouchGFXにあるすべてのUIコンポーネントに関する情報が含まれています。
  • シナリオ - 開発者が遭遇する可能性のあるさまざまなシナリオとその解決方法が含まれています。
