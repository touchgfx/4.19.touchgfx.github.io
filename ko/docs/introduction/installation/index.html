Installation

TouchGFX는 다음과 같은 구성 요소를 포함하는 X-CUBE-TOUCHGFX zip 파일의 형태로 배포됩니다.

  • TouchGFX Designer  - Windows 기반의 GUI 빌더를 통해 UI 빌드
  • TouchGFX Generator - Create a custom TouchGFX HAL through STM32CubeMX
  • TouchGFX Engine - UI 애플리케이션을 구동하는 TouchGFX C++ 프레임워크

STM32 평가 키트에서의 프로토타이핑​

If your intention is to simply try TouchGFX Designer and perhaps do some prototyping on STM32 Evaluation kits, refer to the section Installing TouchGFX Designer. 

TouchGFX Designer 설치​

ST.com 공식 웹사이트에서 하드 드라이브로 X-CUBE-TOUCHGFX를 다운로드한 다음 압축을 해제하십시오.

St.com에서 X-CUBE-TOUCHGFX 다운로드

Inside the extracted folder, you will find the TouchGFX .msi installer in the following path:

Utilities\PC_Software\TouchGFXDesigner

Double-clicking the .msi file will bring up the installer. 지침에 따라 설치 프로세스를 완료하십시오.

TouchGFX Designer 설치

Tip
TouchGFX Designer 내에서 직접 보드를 플래싱하려면 다음과 같은 도구가 필요합니다.
  • STM32CubeProgrammer
  • STM32 ST-LINK 유틸리티

STM32CubeProgrammer 설치​

After downloading STM32CubeProgrammer from STM32CubeProgrammer download location, uncompress the downloaded .zip file and launch the .exe installer file. 지침에 따라 설치 프로세스를 완료하십시오.

TouchGFX Designer 설치

Caution
TouchGFX Designer와 Makefiles에서 대상 하드웨어를 플래싱하기 위해 STM32CubeProgrammer를 사용하려면 반드시 기본 설치 위치에 설치해야 합니다.  C:\Program Files\STMicroelectronics\STM32Cube\STM32CubeProgrammer

STM32 ST-LINK 유틸리티 설치​

After downloading STM32 ST-LINK Utility from STM32 ST-LINK utility location, simply open the downloaded .exe file and follow the instructions to complete the installation process.

STM32 ST-LINK 유틸리티 설치

Caution
TouchGFX Designer와 Makefiles에서 대상 하드웨어를 플래싱하기 위해 STM32 ST-LINK 유틸리티를 사용하려면 반드시 기본 설치 위치에 설치해야 합니다. C:\Program Files (x86)\STMicroelectronics\STM32 ST-LINK Utility

자체 제작 제품 개발​

If your intention is to run TouchGFX applications on either ST display kits or your own custom STM32 based platform, refer to the section Installing TouchGFX Generator in STM32CubeMX.

Installing TouchGFX Generator in STM32CubeMX​

To install X-CUBE-TOUCHGFX through STM32CubeMX, start by selecting "Manage Embedded Software Packages" under "Help" (or press Alt + U).

Help -> Manage embedded software packages

가용 패키지의 최신 목록을 보려면 'Refresh'를 클릭하십시오.

Refresh를 통해 가용 패키지 업데이트

"STMicroelectronics" 탭으로 이동합니다. 화면을 스크롤하여 'X-CUBE-TOUCHGFX'를 찾은 다음 노드를 확장합니다. "TouchGFX Generator"의 확인란을 선택하고 "Install Now"를 클릭합니다. 이제 패키지가 다운로드되고 라이선스 계약이 나타납니다.

TouchGFX Generator 패키지 찾기

Read and accept the license agreement and click "Finish" to install TouchGFX Generator inside STM32CubeMX.

라이선스 계약 수락

The X-CUBE-TOUCHGFX distribution is now unpacked to the following path: 

C:\Users\<user>\STM32Cube\Repository\Packs\STMicroelectronics\X-CUBE-TOUCHGFX\4.13.0
