TouchGFX Simulator Features​

Simulator​

!!crwdBlockTags_40_sgaTkcolBdwrc!!Enables/disables display of pointer coordinates as well as RGB color of the pixel at that coordinate (in hexadecimal).
!!crwdBlockTags_41_sgaTkcolBdwrc!!Enables/disables highlighting invalidated area
!!crwdBlockTags_42_sgaTkcolBdwrc!!Takes a screenshot and places the image under the screenshots folder
!!crwdBlockTags_43_sgaTkcolBdwrc!!Takes screenshots of the next 50 frames and places the images under the screenshots folder
!!crwdBlockTags_44_sgaTkcolBdwrc!!Takes a screenshot and places it in your clipboard
!!crwdBlockTags_45_sgaTkcolBdwrc!!If a simulator skin is used - enables/disables the simulator skin
If a simulator skin is!!crwdBlockTags_46_sgaTkcolBdwrc!! used - enables/disables window border
!!crwdBlockTags_47_sgaTkcolBdwrc!!Sends the application straight back to the startup screen by calling FrontendApplication::changeToStartScreen().
!!crwdBlockTags_48_sgaTkcolBdwrc!!Pauses the simulator by preventing ticks to be sent to the application. Pressing F9 again will resume normal execution.
!!crwdBlockTags_49_sgaTkcolBdwrc!!While the simulator is paused (after pressing F9) it is possible to send a single tick to the application by pressing F10 thereby "single step" the application.
